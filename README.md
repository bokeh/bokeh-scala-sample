# bokeh-scala-sample

Sample application showing integration with [bokeh-scala][bokeh-scala].

## Usage

```bash
$ git clone https://github.com/bokeh/bokeh-scala-sample.git
$ cd bokeh-scala-sample
$ ./sbt run
```
